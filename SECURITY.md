# Security Policy

## Reporting a vulnerability

Please do **not** open a public issue for security vulnerabilities.

Instead, email the maintainers privately at
`hello@generalist.dev` with:

- A description of the issue
- Steps to reproduce
- Impact assessment
- Any suggested mitigation

We will acknowledge receipt within a reasonable timeframe and work with
you on a coordinated disclosure and fix.

## Scope

In scope:

- The Forge binary and Go packages in this repository
- Documented configuration surfaces

Out of scope:

- Issues in upstream model providers (Anthropic, OpenAI, etc.)
- Third-party plugins or custom tools not shipped in this repo
- Social engineering of maintainers
